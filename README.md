📘 Java Core & OOP Practice Repository
📌 Overview

This repository contains structured Java programs organized into Day-wise learning modules:

Day 1: Core Java Concepts
Day 2: Object-Oriented Programming (OOP)

The goal is to build a strong foundation in Java, starting from syntax and control concepts to full OOP implementation.

Constructors
Control over program structure

🔵 Day 2: OOP Concepts

Static and Final keywords
Enum usage
